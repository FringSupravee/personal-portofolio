Project Overview

Date: Dec 2021 - Jan 2022
Tool: Figma
Position: Researcher, Designer

Moneyshare is an application that eliminates the process of a group's bill splitting. The idea occurs when eating out with friends, and everyone eats different share for different dishes. Initially, the concept was already created, and I have seen the web application demo. I was approached to help beautify the interface. Still, after I started doing research, I decided that the application needed a make-over and a better user experience to satisfy users.

5/7 Ideate & Sketch

Core Feature
Based on research insights and discuss with developer, I have listed the features for this project...
  • Scan the bill: I got an idea from Splitwise that allows the premium package to scan receipt and itemization. Instead of manually fill the bill information. Users can open the camera and use AI to automatically load. User only needs to select payer and payee within a few seconds.
  • Auto-check: Add auto-generate for the payment to check the bill's total cost on a web application is similar to a bill that the user received from a shop or restaurant.
  • Notify: Function to notify members of the group about any activities within the group.
  • Request to edit: Allow users to edit the transaction by sending the request to an account that created the bill. Because we want to prevent people from adding false information if allowing everyone to edit freely.
  • Spending tracker: Track all the transactions based on customized periods and visualize them in a pie chart or bar graph. This can benefit the application for linking the information to online banking applications or spending tracker applications in the future.
  • Favorite: Allow users to add the favorite group on the home page to search quickly.

After consulting with the project developers, we decided to cut the premium account idea and receipt scanning because they wanted to create this application for a small circle of friends. So monetizing this project and put up on App Store or Play Store is not their goal.
